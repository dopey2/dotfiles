
## Setup

Config lives in `wezterm/linux/wezterm.lua`. Symlink it into place:

```sh
ln -s ~/repos/dotfiles/wezterm/linux/wezterm.lua ~/.config/wezterm/wezterm.lua
```

---

## Key bindings 

| Key               | Action                                      |
|-------------------|---------------------------------------------|
| CTRL + SHIFT + E  | Split panel vertical (one left one right)   |
| CTRL + SHIFT + O  | Split panel horizontal (one top one bottom) |
| ALT + ARROW       | Panel navigation                            |
| CTRL + LEFT/RIGHT | Jump to words                               |
| HOME/END          | Jump to begin/end of the line               |
